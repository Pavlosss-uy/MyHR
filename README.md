## Overview

**MyHR** is a small, developer-friendly project intended to centralize common HR tasks in one place—so you spend less time juggling spreadsheets and more time supporting your people.

Typical goals include:

- Keep employee records organized
- Track basic HR requests and updates
- Provide a simple UI for day-to-day HR operations
- Enable an API/backend for automation and integrations

> If you’re evaluating this repository: the focus is on clarity, simplicity, and an easy path to extend.

---

## Features

- **People directory**: store and browse employee information
- **Search & filters**: quickly find people and records
- **HR workflow starter**: a base for requests like onboarding, offboarding, and profile updates
- **Backend + frontend** structure: Python for server-side logic and JavaScript for the UI
